# Vue 3 渐进式学习项目

## 项目概况
- 用户是 Vue 3 初学者，正在通过渐进式重构学习企业级开发实践
- 业务场景：登录 → 主页 → 子页面(仪表盘/设置) → 404
- 技术栈：Vue 3 + TypeScript + Vite + vue-router + pinia
- 学习笔记文件：`LEARNING.md`（所有阶段内容追加到此文件）

## 学习阶段进度
- [x] 阶段一：能跑就行 — router 基础、组件内逻辑、平铺路由
- [x] 阶段二：分离关注点 — Layout 布局、嵌套路由、懒加载、目录规范
- [x] 阶段三：状态管理 + 路由守卫 — pinia store、beforeEach、token
- [ ] 阶段四：请求层封装 + 类型约束 — api 层、拦截器、TS interface
- [ ] 阶段五：组合式函数 + 权限体系 — composables、自定义指令、动态路由

## 用户偏好
- 希望例子业务逻辑尽量简单，专注框架搭建和最佳实践
- 全过程记录在 LEARNING.md 一个文件中方便复习
- 中文沟通
